# Claude Monitor

A lightweight Windows overlay that monitors Claude AI usage and active session status for Claude Code and Claude Desktop in real time.
*Project built with Claude Code*

## How it works

The program runs in the background with a system tray icon. A small panel is displayed in the top-left corner of the screen, showing a colored dot and a progress bar indicating usage against the Claude limit.

Hovering over the panel expands it to show:

- All active sessions (Claude Code in VSCode, CLI, Claude Desktop/Cowork);
- The status of each session with a colored dot;
- Current usage percentage against the 5-hour limit;
- Time remaining until the limit resets.

### Session status

- Green — Idle;
- Yellow — Working;
- Red — Waiting for input.

## How to use

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the program:
   ```bash
   python main.pyw
   ```
   Or simply double-click `run.bat`.

3. The icon will appear in the system tray. Use **Show / Hide** to toggle the overlay, and **Quit** to exit.

> **Note:** The monitor reads the authentication token from `~/.claude/.credentials.json`, generated by Claude Code. Make sure you are authenticated.

## Technologies used

- Python 3.11+;
- Tkinter — overlay GUI;
- pystray — system tray icon;
- Pillow — colored icon generation;
- psutil — active process detection;
- requests — Claude usage API consumption.

## Requirements

- Windows 10/11;
- Python 3.11 or higher;
- Claude Code installed and authenticated.
